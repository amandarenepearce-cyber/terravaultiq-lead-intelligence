LEAD ENGINE V5

WHAT THIS VERSION DOES
- Google-based business discovery
- Missing website detection
- Weak website scoring
- Public email, phone, Facebook, and Instagram pull from websites
- CRM-style columns for outreach
- CSV export
- Paste your Google API key inside the app — no file editing

HOW TO USE
1. Unzip this folder into a normal folder like C:\leadtoolv5
2. Double-click START_LEAD_ENGINE_V5.bat
3. Your browser should open automatically
4. Paste your Google API key into the app
5. Search by business type + city

GOOD FIRST TEST
- Lead type: roofers
- Search word: roofers
- City: Leavenworth, KS
- Radius: 25

IF IT OPENS A BLANK PAGE
Use http://localhost:8501 in your browser.
